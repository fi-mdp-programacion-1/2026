# Programación — FI UNMDP

Guías de Trabajos Prácticos de las materias de programación de la Facultad de Ingeniería, Universidad Nacional de Mar del Plata.

## 📂 Contenido

El repositorio contiene las guías de ejercicios de dos materias, organizadas por unidad temática. Cada guía incluye ejercicios de práctica y ejercicios extraídos de exámenes de años anteriores.

---

### 🔹 [Programación 1](GuiasProgra1/)

Programación en **C**: desde los fundamentos de la informática hasta manejo de archivos y estructuras de datos. *(Programación Estructurada)*

| Guía | Tema | Contenido |
|------|------|-----------|
| [Guía 01](Guia_01.md) | Introducción a la Informática | Historia de la computación, generaciones, arquitectura básica |
| [Guía 02](Guia_02.md) | Tipos de datos y E/S | Tipos de datos en C, `scanf`/`printf`, casting, operadores bit a bit |
| [Guía 03](Guia_03.md) | Funciones y ámbito | Ámbito de variables, `static`, paso por valor y referencia |
| [Guía 03r](Guia_03.md) | Recursividad |  |
| [Guía 04](Guia_04.md) | Estructuras de control | `if`/`else`, `switch`, bucles, teorema de Böhm-Jacopini |
| [Guía 05](Guia_05.md) | Arreglos y matrices | Arrays, operaciones sobre vectores y matrices, imágenes |
| [Guía 06](Guia_06.md) | Punteros y memoria dinámica | Punteros, strings, `malloc`/`calloc`/`realloc`/`free` |
| [Guía 07](Guia_07.md) | Archivos | Archivos de texto y binarios, `fopen`/`fclose`/`fread`/`fwrite` |
| [Guía 07b](Guia_07b_StructsUnions.md) | Structs, Unions y Typedef | `struct`, `union`, `typedef`, punteros a estructuras |
| [Guía 08](Guia_08_Integracion.md) | Integración | Exámenes completos 2022–2024 |

---

## 🛠️ Tecnologías

- **Programación 1:** C (estándar C99/C11)
- **IDE sugerido:** Code::Blocks / cualquier IDE compatible con C/C++

## 📝 Estructura de cada guía

Cada guía incluye:

1. **Preguntas orientadoras** — conceptos teóricos para repasar antes de resolver los ejercicios.
2. **Ejercicios de práctica** — problemas progresivos para implementar.
3. **Ejercicios de exámenes anteriores** — enunciados reales de parciales y recuperatorios.
